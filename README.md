# Weather-or-Not
================

Goal: Use the OpenWeatherMap API to build an application.

## Getting Started
1. Install grunt dependencies
	
	```
	npm install
	```
2. Update package.json
	* Update your Name, Description, Repository, Homepage etc.

3. Install bower packages. 

	```
	bower install
	```
4. Run Grunt
	
	``` 
	grunt
	```
